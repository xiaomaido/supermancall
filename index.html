前言
这门课程，源自2016年12月23日0点08分的一条感谢信：“今天收到网易游戏的实习offer了，真的非常谢谢你，你的一个分享，帮了我非常大的忙，给我指明了一条清晰可行的路。能拿到这个实习我非常开心。再次阿里嘎多！”
我不是最顶尖的设计师，也不是最狂热的爱好者，只是多了几年经验的熟练工。曾经也质疑过，为什么设计的教育和学习方法都如此模糊，像一场玄学的探索。理想中的设计方法是对要素进行拆解、分析、重构从而掌握的客观规律。而这些客观基础，使我们避免生产吓人的视觉辣鸡，并决定了我们设计水平天花板的位置。
恰巧这些是我擅长并可以分享给新人的知识。一次对粗糙和丑陋全面否定的态度传递！

一、三观培养
学习先得有宏观意识，才能在微观的决策里有头绪思路。在这一章，学会做一名设计师所该具备的基础素养，对我们的职业有个基本又全面的了解。
学前说明
往期课程回顾，本期的课程内容、方式说明，顺便参点鸡汤
美术史简述
懂点艺术史，以及艺术和设计的关系扫盲，远离肤浅
IT 企业介绍
互联网公司的运作方式以及 UI 设计师的日常和工作内容
职业习惯的培养
一个优秀的设计师，需要一开始就应该养成的良好的职业习惯详解
二、核心基础
UI 设计师，水平的天花板是由各项基础能力所组成的，本章提炼了我认为最重要的基础类型，后面的所有的课程也将围绕它们展开。
平面设计要素
亲密、对齐、对比、重复，黄金比例和几何分割讲解
字体的设计
中英文字体基础认识，初级字体设计方法，如何快速把握要点
栅格与排版
板式设计，辅助线的使用和案例，稳定的页面结构构建
光与色
光是如何形成色彩的，色彩模式的原理和实用的配色技巧
显示器与分辨率
分辨率的详解，设计的标注、切图，以及和前端开发间的沟通方式
路径与曲线
钢笔工具的使用，掌握路径、贝塞尔曲线的奥秘
三、设计日常
设计和学习其它技能一样，需要由易到难，除了设计成套的应用，我们也还有很多其它的设计工作要面对，本章从简单的设计需求开始，掌握初步的工作技能。
设计一套icon
用精密、科学的方式，绘制一套规范的 icon
设计拟物图形 1
学习拟物设计的入门，拟物基础要素的掌握，学会发现细节
设计拟物图形 2
拟物设计的进阶，应对更复杂的场景，学会如何驾驭细节
设计一张 Banner
如何解决 UI 设计师最头疼的设计需求，有效的 Banner 设计方式
设计产品主页
目前主流的产品介绍、下载页面设计方式，入门的网页设计
设计一套幻灯片
集合所有前面学习的知识点，学习模块化的设计思维方式
四、Sketch 的学习
为什么现在学习 UI 设计必须用 Mac 系统，因为有面向这个行业的利器 Sketch，学会使用这个简单的工具，解放我们的生产力。
Sketch 基本功能
Sketch 基本的功能讲解，以及一些总结的小技巧
用 Sketch 设计原型
从原型开始上手 Sketch 的使用，以及用 Flinto 实现交互效果
五、APP 项目设计
本章开始进入正式的项目设计环节。从一个APP立项到收尾我们应该注意哪些问题，使用什么样的工作流程，以及具体设计APP的方法。
启动准备
开始进入具体设计前，应该进行的规划和准备工作
确定风格
设计关键页面，标准化相关的设计元素
设计次要页面
定义好设计元素以后如何快速完成次要页面的设计
规范的整理
根据已经设计好的页面，整理控件和规范
APP 展示设计 1
APP 项目在设计完成以后如何进行展示，标准的做法和流程
APP 展示设计 2
目前主流地产品介绍、下载页面设计方式，入门的网页设计
六、管理界面设计
相比较APP而言，管理系统一般会包含更复杂的操作和结构。通过在这种情况下进一步深入对于设计流程和规范的理解，会帮助我们更游刃有余的完成其它设计工作。
相关介绍
管理界面是什么，设计上有什么不一样的标准和设计方法
前期准备
确定需要设计的原型、模块，以及设计关键页面的风格
绘制剩余页面
将风格量化成组件，如何快速推进到复杂的系统中去
管理界面展示设计
界面设计的展示，为我们的项目增加观赏性
七、学习结尾
最后一章，学会怎么准备设计师的简历，相关面试的技巧。并对前面所有学习的知识点进行回顾，在课程结束后我们该如何使用学过的知识进行自我提升，有效率的自学。
简历与面试
该怎么做设计师的简历，当我们面试的时候我们在讨论什么
课程总结
回顾本期课程学过的知识点，一起展望未来
